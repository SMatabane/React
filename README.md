# Scoop & Smiles Ice Cream Shop Website
This project is a React-based web application designed to showcase the delightful variety of ice cream items available at Scoop & Smiles, a fictional ice cream shop.

## Technologies Used
React.js
HTML5
CSS3
JavaScript
